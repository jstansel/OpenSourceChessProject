Kings & Queens

Welcome to a work in progress.

This is a chess application designed to run over the Internet using standard web browsers. It is anticipated this application will be compatible with any device supporting a standard browser.

The functionality provided will allow chess games to be played between two players while simultaneously being viewed by multiple spectators. The moves for the game will be recorded, the ability to down-load the moves from the in standard chess formats is also available. The application will validate all moves as well as the correct flow of the game.
